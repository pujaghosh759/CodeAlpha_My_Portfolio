#  Personal Portfolio Website

This project is a fully responsive **Portfolio Website** built using **HTML, CSS**, and **JavaScript** (optional effects).  
It showcases personal information, skills, projects, and contact details with smooth animations and a modern UI.

---

##  Features

### Clean & Modern Design  
- Simple, elegant layout  
- Smooth scrolling  
- Hover animation effects  
- Section-based navigation  

###  Fully Responsive  
- Works perfectly on **mobile**, **tablet**, and **desktop**  
- Flexible layout using CSS Grid & Flexbox  

###  Portfolio Sections  
- **Home / Hero section**  
- **About Me**  
- **Skills**  
- **Projects**  
- **Resume / Download CV**  
- **Contact Form / Social Links**

###  JavaScript Enhancements  
- Smooth scroll  
- Menu toggle (mobile view)  
- Fade / slide animations  
- Active menu highlight
